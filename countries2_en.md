#Assignments

Write a stream pipeline that:

Returns whether there is at least one country with the word island in its name ignoring case.

Returns the first country name that contains the word island ignoring case.

Prints each country name in which the first and the last letters are the same ignoring case.

Prints the populations of the first ten least populous countries (required intermediate operation: stream, limit)).

Prints the names of the first ten least populous countries (required intermediate operation: sorted, limit)).

Returns summary statistics about the number of country name translations associated with each country (required intermediate operation: mapToInt, summaryStatistics).

Prints the names of countries in the ascending order of the number of timezones.

Prints the number of timezones for each country in the form name: population, in the ascending order of the number of timezones.

Returns the number of countries with no Spanish country name translation (the Spanish language is identified by the language code es).

Prints the names of countries with null area.

Prints all distinct language tags of country name translations sorted in alphabetical order (required intermediate operation: flatMap).

Returns the average length of country names.

Prints all distinct regions of the countries with null area.

Returns the largest country with non-null area.

Prints the names of countries with a non-null area below 10 (requires the use of BigDecimal.TEN).

Prints all distinct timezones of European and Asian countries.
